# WebApp

Node project


## Installation Details
---
### Locally
*Install Node*
> sudo apt-get install curl
> curl -sL https://deb.nodesource.com/setup_13.x | sudo -E bash -
> sudo apt-get install nodejs

Check NodeJs and npm version
> node -v

npm
> npm -v

*Install all package dependencies*
> npm ci

*Start application*
> npm start

*Run unit tests*
> npm test
